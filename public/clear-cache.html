🔄 清除缓存工具 / Clear Cache Tool

问题描述 / Problem Description

如果您看到旧的放射状模式（3个橙色圆圈）而不是新的13个超星系团，这是因为浏览器缓存了旧数据。

If you see the old radial pattern (3 orange circles) instead of the new 13 superclusters, it's because your browser cached the old data.

方法1：自动清除 / Method 1: Automatic Clear

点击下面的按钮清除缓存并重新加载数据：

Click the button below to clear cache and reload data:

方法2：手动硬刷新 / Method 2: Manual Hard Refresh

Windows/Linux: Ctrl + Shift + R 或 Ctrl + F5

Mac: Cmd + Shift + R

方法3：清除浏览器缓存 / Method 3: Clear Browser Cache

Chrome / Edge:

  1. 按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)
  2. 选择"缓存的图片和文件" / Select "Cached images and files"
  3. 时间范围选择"全部时间" / Time range: "All time"
  4. 点击"清除数据" / Click "Clear data"

Firefox:

  1. 按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)
  2. 勾选"缓存" / Check "Cache"
  3. 点击"立即清除" / Click "Clear Now"

验证数据已加载 / Verify Data Loaded

打开浏览器控制台（按 F12），查找以下日志：

Open browser console (press F12), look for these logs:

[UniverseDataLoader] Loaded /data/universe/laniakea.bin: 169984 bytes
[UniverseDataLoader] Supercluster count: 13
[UniverseDataLoader] Parsed 13 superclusters, 14096 total galaxies
[Laniakea] Loading data: 13 superclusters, 14096 galaxies

✅ 如果看到这些数字，说明数据已正确加载！

✅ If you see these numbers, the data is loaded correctly!
